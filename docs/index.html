You can zoom and pan the scatter plot.

Flow:

  1. Click on the scatter plot to add points.
  2. Click "Solve For Params" to see what parameters create the linear regression line of best fit.
  3. Change the dropdown to "Logistic Regression" to split the points into two classes - red and blue. Initially, points above y=x are made to be blue and points below y=x are red. You can click to add more points and can use the drop down to place blue or red points. The objective is now to classify the points as either red or blue.
  4. Click "Solve For Params" to see the logistic regression classification boundary as it runs gradient descent for 8 seconds.
  5. You can add more points and hit "Solve For Params" again once the boundary line has stopped moving.
  6. Switch to "Support Vector Machine (SVM)" and hit "Solve For Params" to run 8 seconds of gradient descent for that model.
