You can zoom and pan the scatter plot.

Flow:

  1. Click on the scatter plot to add points.
  2. Click "Solve For Params" to see what parameters create the linear regression line of best fit.
  3. Change the dropdown to "Logistic Regression" to split the points into two classes - red and blue. The objective is now to classify the points as either red or blue.
  4. Click "Solve For Params" to see what parameters create the logistic regression classification boundary.
